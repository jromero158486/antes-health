# ANTES Health multipage website

Static GitHub Pages website for ANTES Health.

## Pages
- `index.html` — Home
- `story.html` — Our Story
- `mission.html` — Mission & Vision
- `approach.html` — Our Approach with evidence cards and animated data-flow map
- `pilot.html` — MVP Pilot
- `awards.html` — Awards & Milestones
- `contact.html` — Join Waitlist / Contact
- `partners.html` — For Partners
- `investors.html` — For Investors
- `faq.html` — FAQ
- `privacy.html` — Privacy & Data Protection
- `medical-disclaimer.html` — Medical Disclaimer

## Before publishing
1. The waitlist form in `contact.html` uses FormSubmit with `joselynromero9@gmail.com`. After the first test submission, confirm the activation email from FormSubmit so submissions forward correctly.
2. Awards are displayed as custom logo-style SVG badges in `assets/`. They are not official logos. Replace them with official logo files only if you have permission to use each brand asset. Claude Startups is marked as accepted.
3. Add a real founder photo if desired and replace the placeholder in `story.html`.
4. Keep medical safety wording: ANTES Health does not diagnose, treat, replace clinicians, or provide emergency care.

## GitHub Pages
Upload all files to your `antes-health` repository and enable GitHub Pages from the main branch/root folder.
